# Product

| Item | Definição |
| --- | --- |
| Produto | Sistema de agendamento de confissões para uma igreja. |
| Público | Fiel sem login; secretaria; padre; administrador. |
| Objetivo | Reserva privada, simples e sem conflitos. |
| Canal | Web mobile-first; QR Code público como acesso facilitado. |
| Privacidade | Não coletar conteúdo espiritual; existência do agendamento é privada. |
| Fonte oficial | `.ai/docs/produto-mvp.md`. |

## MVP

- Selecionar horário; agendar; código privado; comprovante opcional.
- Consultar, alterar, cancelar e recuperar código por fluxo privado.
- Agenda/operação interna; configurações; auditoria.
- Gestor: visualizar QR Code público, baixar PNG/PDF e regenerar.

## Fora do MVP

- Login de fiel; escolha pública de padre; listas públicas; notificações; pagamentos; multiunidade; lista de espera.
